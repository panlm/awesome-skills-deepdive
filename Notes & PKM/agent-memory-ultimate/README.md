# 🧠 Agent Memory Ultimate — 生产级代理记忆系统

## 描述

一个生产就绪的记忆系统，支持每日日志、睡眠整合、SQLite + FTS5 全文搜索，以及 WhatsApp/ChatGPT/VCF 数据导入器。

## 功能特点

- **每日日志**：自动化的每日记忆记录
- **睡眠整合**：模拟人类记忆整合过程的休眠期数据处理
- **SQLite + FTS5**：使用 SQLite 全文搜索引擎进行高效记忆检索
- **多源导入**：支持从 WhatsApp、ChatGPT、VCF 联系人文件导入数据
- **持久化存储**：跨会话的持久记忆存储
