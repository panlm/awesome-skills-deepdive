# guardskills

## 标题和描述

**guardskills** — 一个 OpenClaw 技能安装安全门禁工具。

- **作者**: @felixondesk
- **分类**: Git & GitHub
- **安装**: `clawhub install felixondesk/guardskills`

## 功能特点

- 作为技能安装的安全检查门禁
- 在安装新技能前进行安全评估
- 帮助过滤潜在的恶意或不安全技能

## 使用方法/示例

```bash
# 安装
clawhub install felixondesk/guardskills

# 或使用 npx
npx clawhub@latest install felixondesk/guardskills
```

安装后，该技能将在安装其他技能前进行安全检查，作为安装流程的安全门禁。

## 安全评估

| 评估项目 | 状态 | 说明 |
|---------|------|------|
| 源码可用性 | ⚠️ 不可用 | GitHub 仓库返回 404，源代码不可审计 |
| 下载量 | 🔴 零 | 0 次下载，0 次安装 |
| 权限范围 | ❓ 未知 | 无法审查源码确认权限需求 |
| 数据安全 | ❓ 未知 | 无法确认是否收集或外泄数据 |
| 总体评级 | 🔴 可疑 | 源码缺失，无法进行安全审计，不建议安装 |

**⚠️ 警告**: 此技能声称是"安全门禁"工具，但其自身源代码在 GitHub 仓库中不可用（404）。一个安全工具自身无法被审计，这是严重的信任问题。
