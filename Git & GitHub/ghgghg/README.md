# ghgghg

## 标题和描述

**ghgghg** — 一个 OpenClaw 技能，用于查询 GitHub 仓库的 Star 数量和代码行数。

- **作者**: @chenpinji
- **分类**: Git & GitHub
- **安装**: `clawhub install chenpinji/ghgghg`

## 功能特点

- 查询指定 GitHub 仓库的 Star（收藏）数量
- 统计仓库的代码行数
- 简单直接的 GitHub 仓库信息查询工具

## 使用方法/示例

```bash
# 安装
clawhub install chenpinji/ghgghg

# 或使用 npx
npx clawhub@latest install chenpinji/ghgghg
```

安装后，可通过 OpenClaw 智能体查询任意 GitHub 仓库的 Star 数和代码行数。
