## 功能概述
- 支持长度单位换算（厘米、米、英尺、英里、公里等）
- 支持重量单位换算（克、千克、磅、盎司等）
- 支持温度换算（摄氏度、华氏度、开尔文）
- 支持面积单位换算（平方米、平方英尺、英亩等）
- 支持体积单位换算（毫升、升、加仑等）
- 支持速度和时间单位换算
- 支持数据存储单位换算（字节、KB、MB、GB 等）
- 提供单位列表查询和类别浏览功能

## 使用场景
- 日常工作中需要在公制和英制单位之间快速转换
- 开发过程中处理数据存储容量计算
- 跨国协作时处理不同计量体系的换算
