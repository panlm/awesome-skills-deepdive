# git-sentinel

## 标题和描述

**git-sentinel** — 一个 OpenClaw 技能，让智能体扮演高级软件工程师和安全审计员的角色。

- **作者**: @corezip
- **分类**: Git & GitHub
- **安装**: `clawhub install corezip/git-sentinel`

## 功能特点

- 让 AI 智能体以高级软件工程师身份审查代码
- 安全审计功能，识别代码中的安全隐患
- 结合 Git/GitHub 工作流进行代码审查

## 使用方法/示例

```bash
# 安装
clawhub install corezip/git-sentinel

# 或使用 npx
npx clawhub@latest install corezip/git-sentinel
```

安装后，智能体将以高级软件工程师和安全审计员的身份对代码进行审查和评估。

## 安全评估

| 评估项目 | 状态 | 说明 |
|---------|------|------|
| 源码可用性 | ⚠️ 不可用 | GitHub 仓库返回 404，源代码不可审计 |
| 下载量 | 🔴 零 | 0 次下载，0 次安装 |
| 权限范围 | ❓ 未知 | 无法审查源码确认权限需求 |
| 数据安全 | ❓ 未知 | 无法确认是否收集或外泄数据 |
| 总体评级 | 🔴 可疑 | 源码缺失，无法进行安全审计，不建议安装 |

**⚠️ 警告**: 此技能的源代码在 GitHub 仓库中不存在（404），存在严重的安全审计障碍。"安全审计"类工具如果自身不可审计，风险尤其高。
