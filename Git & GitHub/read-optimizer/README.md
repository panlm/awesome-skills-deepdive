## 功能概述
- Smart Read 模式：同时读取文件头部和尾部指定行数，适合日志和大型文档
- Grep Read 模式：按正则表达式匹配读取特定行，支持上下文行数控制
- Diff Read 模式：仅读取自上次 Git 提交以来的变更行
- 可配置参数：文件路径、模式、行数、匹配模式、上下文行数

## 使用场景
- 高效检查大型日志文件，只看头尾关键信息而不加载全文
- 在代码库中按错误关键词快速定位问题行
- 查看文件自上次提交以来的变更内容

## 依赖和前提条件
- Node.js
- Git（Diff 模式需要在 Git 仓库中使用）
