# 💓 Heartbeat Checklist — 定期健康检查清单

## 描述

一个定期运行的检查清单技能，建议每日运行 2-4 次，用于系统化地进行健康和状态检查。

## 功能特点

- **定期检查**：支持每日 2-4 次的定期运行
- **清单式管理**：系统化的检查流程
