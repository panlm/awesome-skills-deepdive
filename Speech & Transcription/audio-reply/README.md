## 🔍 功能特点

该技能使用 MLX Audio TTS (chatterbox-turbo 模型) 生成语音回复。

**触发短语：**
- "read it to me [URL]" — 获取公共网页内容并朗读
- "talk to me [topic/question]" — 生成对话式语音回复
- "speak"、"say it"、"voice reply" — 将回复转换为音频

**安全限制：**
- 仅获取 http:// 或 https:// URL
- 需要 macOS (Apple Silicon) 和 uv 工具
